# 🏧 Sistema de Saque Automático / Automatic Teller Machine (ATM)

| 🇧🇷 Português | 🇺🇸 English |
| :--- | :--- |
| **Descrição** | **Description** |
| Este é um aplicativo de console desenvolvido em **C#** que simula um caixa eletrônico. O sistema calcula a menor quantidade de cédulas para um saque. | This is a **C#** console application that simulates an ATM. The system calculates the minimum number of banknotes required for a withdrawal. |
| **Funcionalidades** | **Features** |
| - Validação de entrada (impede letras e valores ≤ 0).<br>- Cálculo otimizado para notas de R$50, R$20, R$10, R$5 e R$1.<br>- Interface colorida para melhor experiência. | - Input validation (prevents letters and values ≤ 0).<br>- Optimized calculation for $50, $20, $10, $5, and $1 bills.<br>- Colored interface for better user experience. |
| **Tecnologias** | **Technologies** |
| C#, .NET SDK, Git/GitHub. | C#, .NET SDK, Git/GitHub. |

## 📋 Lógica de Programação / Programming Logic

| 🇧🇷 PT-BR | 🇺🇸 EN-US |
| :--- | :--- |
| O sistema utiliza os operadores de divisão (`/`) e resto (`%`) para distribuir o valor entre as cédulas disponíveis, garantindo que o usuário receba o menor número de notas possível. | The system uses division (`/`) and modulo (`%`) operators to distribute the amount among available bills, ensuring the user receives the fewest banknotes possible. |
